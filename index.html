Idle Movie Theatre

Idle Theater is a production ready idle genre game. It features a robust progression system where you start with a basic movie theater which you can then upgrade to build an empire. It has a fledged out in-game economy where you receive money from customers and use that money to upgrade your theaters. It also features rewarded video ads at strategic points to improve conversion rates. The game has different progression values; exponential, linear and variable progression. It features 3 different locations, city, beach and moon. It also has a complex queue system. The build includes multiple different analytics like Facebook and Appmetrica and Ironsource for ad mediation.

Tipsy Ball

Tipsy Ball is a casual arcade gamewhere you have to control a ball and avoid hitting obstacles. The game is dynamic and has a power bar which fills up when you collect gems giving you invincibility for a fixed duration. The levels are generated randomly by code. The colour palette is also chosen at random. The game features the Eazegames SDK that adds in an online multiplayer option where you can log in with your Eazegames account and play for real money and challanges in real time with other players.

Mazerace.io

MazeRace.io was developed with the help of Joel Fernandes from Stareout games. The game features bots that use AI scripts and certain algorithms to race against the main player. The complexity of the AI can be controlled. The game features a lot of different environments. The controls are perfected along with squash animations and other feedbacks to improve 'Game Feel' for the player.

Retro Revolution

Retro Revolution is a a 2D casual arcade game, that I made with a team of 3, for Poco by Xiaomi as part of their Game Development Academy. I worked as the Level Designer and Developer for the game. The game features 3 different locations, a powerup system and a shop to upgrade powers and buy cosmetic items. It also has a Leaderboard and achievements system to keep the player engaged. The game was released on the Google Play Store as well as the Mi App Store.

Archery Trickshots

Archery trickshots uses gravity and height formula to calculate a parabolic path for the arrow to take to hit the target. 


Vector3 velocityY = Vector3.up * Mathf.Sqrt(-2 * gravity * height);
Vector3 velocityXZ = displacementXZ / (Mathf.Sqrt(-2 * height / gravity) +
Mathf.Sqrt(2 *(displacementY - height) / gravity));

This helps to calculate a trajectory for the arrow to follow that adheres to the laws of physics and looks as natural as possible. The game also uses rigged human models with character animation to draw and shoot the bow.

Cool Dunk

Cool Dunk uses bezier curves for accurate ball throws. It also has human rigged models with walk, idle, and celebratory animation cycles. It has ragdoll effects for collisions. The game consists of a full Shop where you can buy different types of balls as cosmetics. The game features different locations for you to play in.

Flippy Driver

Flippy Driver is a 3D racing game where you have to jump off buildings and flip in the air to get speed bonuses. The game heavily relies on Unity physics to simulate real car mechanics and gravity. It features a robust movement function that allows the player to move the car and rotate it in the air smoothly. The camera is controlled by Cinemachine follow script and a lerp amount is applied to smoothen the motion of the camera when the car accelerates and decelerates.

Animix Puzzle

Animix features a dynamic puzzle system. Each of the puzzle is animated with a unique style. A Sprite mask is used to separate and break the animations into distinct blocks. Each part of the animation keeps playing in their own block. The player has to combine the blocks to complete the animated video. Every puzzle has it's own unique animation.

Budget Tracker

Budget Tracker is a web app developed using the MERN stack. It's deployed on heroku and the backend data is stored on MongoDB. It's a simple CRUD app where you can update and delete a list dynamically and the data is stored in a remote server.

Reverse Car

A 3D game where you drive a car but in the reverse direction. It uses the Path Creator tool by Sebastian Lague. The tool uses Bezier curves to form paths with varied meshes. It features a car with working wheel colliders, steering controls, coins and fever mode.

Unity Ml agents

A small project exploring the basic premise of reinforcement learning using Unity's ML agents. The project trains a game object to jump over obstacles at the correct time to score a point. The objective is to create an intelligent player object that can successfully detect an incoming projectile/obstacle and correctly assess the response time it needs to jump over the said projectile/obstacle without any collision.

Fractal Forest

A randomly generated fractal forest that has a different seed each time you refresh the page. The fractals are recursive line objects. The project is made using the p5.js libary that utilises the graphical capabilities of javascript.

Balloon Blaster 2D

A 2D ball blaster game with 2D physics. You fire a canon with limited rounds and try to blow up every ball present in the level. Uses trails, explosive particle effects and effective camera shake to make the action of shooting very satisfying for the player.

Leap of Chimp

A pixel art styled, doodle jump inspired casual game with retro-themed soundtracks and SFX. The game has an infinite mode which allows you to go to space and beyond while you jump on tree branches collecting bananas which are used as in-game currency and avoiding bugs. You also get powerups and 6 different avatars. It featuers Google achievements, leaderboards as well as in-app purchases. It also uses a complex two camera system to simulate a seamless transition from one end of the screen to another for the player.

Gunman Gunther

The first game that I ever released. A very simple infinite arcade game where you jump, shoot birds and avoid incoming obstacles. Features a wild-west retro aesthetic, chiptune music and a leaderboard. Also has Google Admob's banner and reward ads.
